# 一、市场认知层
（他如何理解市场、趋势、情绪、不确定性）

- **理念要点：存量博弈下的“跷跷板效应”与板块轮动**
  - **关键原始表述**：“存量，要习惯上下扫”、“总量不变就是拆东墙补西墙，跷跷板”、“一个板块涨了其他几个板块就会跌”。
  - **讲解/案例说明**：他认为当前市场缺乏场外增量资金（增量要元旦后），表现为大盘股（七姐妹）、明星小盘股、币股三个板块之间的资金轮动。当资金流向小盘股博反弹时，大盘股会受压制；反之亦然。

- **理念要点：先行指标意识（BTC作为市场风向标）**
  - **关键原始表述**：“btc对打仗最敏感”、“btc 和tsll 都是风向标”、“币先行，反弹继续还是回调最准”。
  - **讲解/案例说明**：他极度重视加密货币市场的领先性，认为币市通常比股市提前10-30分钟反应。通过监控BTC/ETH的5分钟涨跌幅（1%预警）来预判大盘及相关个股的入场或撤离时机。

- **理念要点：历史惊人相似的规律性认知**
  - **关键原始表述**：“美股一共就几种图，多经历几个循环就熟悉了”、“和11月14日的会像点”、“1月24日日本加息前也是周四周五跌...和这次一样”。
  - **讲解/案例说明**：他主张建立“赚钱记忆”，将当前的市场行为（如加息前反弹再跌、四巫日双杀、政府关门预期）与历史特定日期进行对标，认为市场会不断重复过去的结构，以此作为操作基准。

# 二、操作原则层
（他在什么情况下参与、回避、试错、加减仓）

- **理念要点：确定性套利（跌幅的一半/前高回归法则）**
  - **关键原始表述**：“100跌50的明星股，保守回75一半位置，到100你50买的就是翻倍”、“反弹一半位置出一半，剩下博前高”。
  - **讲解/案例说明**：他筛选腰斩的明星股，利用数学概率博取反弹空间。案例中如APLD、INTC、RKLB均遵循“低位建仓—回升至跌幅50%处止盈一半—目标前高”的节奏。

- **理念要点：利空消化的“三日法则”与“转弯”确认**
  - **关键原始表述**：“资讯一增发了，等明天三天消化”、“不要一利空就马上进”、“分时有个尖角就是转弯了”。
  - **讲解/案例说明**：对于增发、高管减持或财报利空，他主张分三日观察，不急于接飞刀。入场时必须看到分时线上的“转弯”（2分钟内涨/跌幅达1%，且成交量覆盖前一段卖盘/买盘）。

- **理念要点：事件驱动的操作节奏（加息/利率会议）**
  - **关键原始表述**：“加息前反弹是预防性的，先升后跌”、“靴子落地就是利好”、“日本加息清杠杆，周四前减到3成”。
  - **讲解/案例说明**：他主张在重大风险事件前利用预期的反弹“倒T”减仓（套现现金），等待事件正式落地（跌到低位确认支撑）后，再回补仓位做“正T”。

# 三、风控与纪律
（仓位、止损、节奏、容错、心态要求）

- **理念要点：复合式仓位策略（死拿+波段）**
  - **关键原始表述**：“一份死拿，一份做T”、“低位买的一半不动到前面跌下来一半减一半...剩下博前高”。
  - **讲解/案例说明**：通过“九成拿、一成T”或“五五开”的配置，既保留长线仓位防止卖飞，又通过日内T降低持仓成本，应对震荡市的心理压力。

- **理念要点：期权的严格纪律与时间成本管理**
  - **关键原始表述**：“长期权短做”、“不能像知道加息利空硬抗，股票可以，期权不行”、“-38%通用止损”。
  - **讲解/案例说明**：他深知期权的时间衰减（Theta）和高IV陷阱。主张买入长期权（如4月以后）但在日内异动时离场，拒绝在重大不确定性（如日本加息）中硬抗期权磨损。

- **理念要点：节点避险与现金流意识**
  - **关键原始表述**：“18-19号规避日本加息”、“圣诞元旦有基金现金流要求”、“收盘留8成过周末”。
  - **讲解/案例说明**：强调在机构因节日赎回或政策结算可能导致流动性缩减的节点前，主动降低仓位，持有现金以换取后续底部买入的权利。

# 四、反复出现的底层逻辑
（多次出现、核心信念）

- **底层逻辑概括：量能决定论（只看钱，不看消息）**
- **对应多次原始表述**：“量化不看消息，就看资金成交额和转弯时候的资金额”、“抓反转永远是大资金先手权利”、“看成交额是不是把抛的都接回”。
- **统一解释**：他认为所有的消息（H200放开、合作伙伴取消等）只是机构砸盘或拉升的借口。操作的核心应是计算分时图中的成交额对比，判断买盘是否足以扭转趋势。

- **底层逻辑概括：存量市场的“追跌杀涨”反向思维**
- **对应多次原始表述**：“存量，追跌杀涨”、“只有增量可以追高”、“顶不上去都会下去测试”。
- **统一解释**：在存量环境下，价格上涨即意味着抛压积累，下跌反而创造安全边际。因此操作上必须在支撑位买入，阻力位坚决减仓。

# 五、本阶段风格标签（辅助理解）
- **风格关键词**：量化对标、反向博弈、波段为王。
- **风险偏好判断**：中性偏审慎。极度厌恶在高位被动持仓，尤其在重大政策变动（日本加息）前具有极高的警觉性。
- **操作节奏特征**：高频波段。擅长利用盘前人工干预和盘中强平时间点进行“三段式”套利，重视日内转折点的捕捉。
