## 核心总结
- 过去 24 小时内，这位博主的研究重点依然集中在三条主线：**Neoclouds（对应能源/算力基础设施）**、**Memory（存储）**、**Photonics（光子/光通信）**。
- 他强调，过去几个月市场曾普遍看空存储板块，但相关标的随后大幅上涨，借此表达对**存储产业链持续强势**的认可，并暗示市场此前明显低估了该方向。
- 他提到自己核心主题中选中了各细分赛道的“头部表现者”，包括 **$NBIS、$EWY leaps、$SIVE**，显示其当前仍以这些方向作为主要研究/交易框架。
- 他在寻找一类机会：**NAV 折价明显、但又独立具备 AI 增长逻辑的公司**，并明确表示自己已持有 **$ACMR** 和 **WUS (2316)**，希望继续挖掘类似机会。
- 他回顾了自己此前对 **MLCC 瓶颈**的观察，认为这类“产业瓶颈”往往对应做多机会；相关受益股近两个月表现很强，包括 **Taiyo Yuden、$VSH、Murata**。
- 对 **Intel + Apple** 相关新闻，他更多是在调侃由 Trump 宣布合作消息推动 **$INTC** 单日上涨 8%，并指出连 Intel 管理层都“意外”，说明其对该消息的可持续性和严肃性带有一定戏谑/怀疑意味，而不是明确基本面看多。
- 对卖方/舆论层面，他讽刺 **Bernstein** 此前对 **Kioxia** 给出 -50% 目标后股价反而大涨，也再次强调其一贯观点：**市场共识和机构悲观看法在部分科技硬件方向上可能是反向指标**。

## 涉及标的与方向
- **$MU / Micron**：博主用过去 3 个月涨幅举例，强调市场曾普遍看空 memory，但股价大涨，体现其对存储板块中期趋势偏多。理由是板块此前预期过低、实际表现显著超预期。  
- **$SNDK / Sandisk（或相关存储标的）**：与 $MU 一起被作为 memory 反转案例，意在说明存储行业情绪修复和景气上行。方向偏多。  
- **SK Hynix / Samsung**：作为韩国存储龙头，被列入过去 3 个月显著上涨名单，强化其对 memory 主线的看多框架。  
- **$EWY**：被博主纳入 memory 相关收益表达中，并提到自己做了 **$EWY leaps**，说明其不仅看个股，也通过韩国敞口表达对存储/半导体链条的看多。方向偏多，且带有仓位/策略信息（leaps 长期看涨期权）。  
- **$INTC / Intel**：因 Trump 宣布 Intel + Apple partnership 消息而单日上涨 8%。博主并未给出明确基本面看多结论，更多是在质疑/调侃消息驱动属性，认为 Trump 像是在“硬抬”股价。  
- **$AAPL / Apple**：出现在 Intel 合作消息中，本帖仅作为事件关联方，无更明确投资判断。  
- **$ACMR / ACM Research**：博主明确表示“Have exposure”，说明已有持仓/敞口。他关注其**NAV 折价+AI 成长属性**，并将其列为自己正在继续研究的机会。方向偏多。  
- **WUS (2316)**：与 $ACMR 并列，被视为 NAV 折价较大且具 AI 增长逻辑的研究对象；博主明确表示已持有敞口。方向偏多。  
- **SK Square**：被拿来作为对照，说明博主在比较不同控股/资产折价型公司的 NAV 折价机会。  
- **$ASML**：提到中国如何“走私”类似校车大小的 ASML 设备，属于对行业/地缘供应链问题的感叹，没有明确交易方向。  
- **Kioxia**：博主批评 Bernstein 曾对其给出 -50% 目标价，但股价随后继续上涨且当天再涨 12%，借此表达对卖方悲观判断的不信任。隐含方向偏多。  
- **$NBIS**：被列为其三大核心主题之一 Neoclouds（Energy）中的代表性赢家，说明其在算力基础设施/能源相关方向上有明确关注和较强信心。  
- **$SIVE**：被列为 Photonics 主线中的代表性赢家，显示博主在光子/光通信方向维持偏多框架。  
- **Taiyo Yuden / $VSH / Murata**：博主回顾自己曾提过 **MLCC bottleneck**，而这些公司近两个月涨幅显著，说明他认为被产业瓶颈卡住的环节值得做多。尤其 **$VSH** 被直接点名为受益者。  
- **$SPCX**：仅出现在对 Jim Cramer 的调侃中，没有形成实质投资分析。

## 催化剂与时间线
- **过去 3 个月**：存储板块大幅上涨，博主用 $MU、$SNDK、$EWY、SK Hynix、Samsung 的涨幅作为证据，说明 memory 是其最重要的验证成功主线之一。
- **当天事件驱动**：Trump 宣布 **Intel + Apple partnership**，带动 **$INTC 当日上涨 8%**。这是典型的政策/消息驱动型催化。
- **当天行情**：Kioxia 在此前遭 Bernstein 看空后继续上涨，且**当天再涨约 12%**，成为“卖方看空失效”的反例。
- **近两个月**：围绕 **MLCC 瓶颈** 的相关受益标的表现强劲，Taiyo Yuden、$VSH、Murata 分别录得大幅上涨，显示“供应瓶颈”是有效的产业链催化观察框架。
- **当前持续跟踪主题**：Neoclouds（Energy）、Memory、Photonics，属于其接下来仍会围绕展开的主线研究方向。
- **当前研究中的新方向**：寻找**NAV 大幅折价 + AI 独立成长性**的公司，说明这可能是他近期新增的选股框架。

## 风险与不确定性
- 博主明确提醒：**“not all my ideas are green, especially on short term timeframes”**，即其观点并非短期都有效，短线波动和回撤风险较大。
- 对 **$INTC** 的上涨，他暗示更像是政治人物喊单/营销推动，而非扎实基本面改善，因此事件驱动的持续性存在疑问。
- 他对社交媒体和新闻聚合中的 **AI 垃圾信息/误导性内容农场** 表示担忧，意味着部分公司消息面可能被噪音污染，信息质量风险较高。
- 关于 **ASML / 中国** 的表述也提示出半导体设备链仍受地缘政治、出口管制与灰色供应链问题影响。
- 对存储、MLCC 等高弹性赛道，虽然博主强调已大涨，但这也隐含了**追高后波动加大**的不确定性；不过这是原帖中的情绪背景，并非他明说的风险控制建议。

## 后续可跟踪事项
- 持续跟踪博主三大主线：**Neoclouds（Energy）/ Memory / Photonics** 是否继续扩展到更多标的。
- 关注其已明确有敞口的 **$ACMR**、**WUS (2316)**，看其是否进一步披露 NAV 折价、AI 增长逻辑或增减仓态度。
- 跟踪 **Memory** 链条是否仍有超预期数据/价格/资本开支催化，尤其是 **$MU、SK Hynix、Samsung、$EWY** 相关表述。
- 继续观察 **MLCC 瓶颈** 是否被他升级为新的独立交易主线，以及是否补充更多受益公司。
- 关注 **$NBIS** 与 **$SIVE** 后续是否被博主进一步展开，因它们被其点名为各自主线中的“最佳表现者”。
- 留意他是否继续寻找更多**NAV 折价 + AI 成长**类型机会，这是本次发帖中较明确的新研究方向。
